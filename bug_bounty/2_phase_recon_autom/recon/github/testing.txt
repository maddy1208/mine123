# GitHub Recon for Bug Bounty

## **Objectives**

1. **Organization Recon** – Finding sensitive data related to an organization.
2. **Employee Profiles** – Identifying developer accounts, leaks, and insider information.

## **What You Can Find During Recon**

* Source Code Leakage
* Passwords
* Tokens
* API Keys
* Usernames
* Sensitive Files
* Internal IPs
* AWS Secrets
* SSH, FTP Passwords

### **Using GitHub Dork Helper**

* Visit: [https://vsec7.github.io](https://vsec7.github.io)

### gitleaks

gitleaks detect --source=<cloned-repo>

##trufflehog:
docker run --rm -it -v "$PWD:/pwd" trufflesecurity/trufflehog:latest github --repo https://github.com/trufflesecurity/test_keys

combine with repo:
trufflehog git https://github.com/trufflesecurity/test_keys --results=verified,unknown

combine with org:
trufflehog github --org=trufflesecurity --results=verified,unknown
