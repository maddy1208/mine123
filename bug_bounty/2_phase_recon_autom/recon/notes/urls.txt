1. GETTING ALL URLS USING TOOLS

   ```
   1. 🔍 **Passive URL Enumeration (Waymore + GAU + Wayback):

   	waymore -i domains.txt -mode U -oU waymore_output.txt  
   	gau example.com | anew gau_urls.txt  
   	waybackurls example.com | anew wayback_urls.txt  
   	urlfinder -d domain.com -o  urlfinder.txt


   2. 🛠 **Active Crawling (Gospider + Katana):

   	gospider -S domains.txt -o gospider_output -d 3 -c 10 --js  
   	katana -u https://example.com -d 5 -o katana1_urls.txt
   	katana -u https://example.com -jc -d 5 -o katana2_urls.txt




   6.Using wayback:
   	       curl -G "https://web.archive.org/cdx/search/cdx/" \
   		  --data-urlencode "url=*.empireflippers.com/*" \
   		  --data-urlencode "collapse=urlkey" \
   		  --data-urlencode "output=text" \
   		  --data-urlencode "fl=original" \
   		  > out.txt
   ```

   ## 📌 \*\*Combine All Found URLs (Avoid Duplicates)#\#

   cat waymore\_output.txt gau\_urls.txt wayback\_urls.txt gospider\_output katana1\_urls.txt katana2\_urls.txt out.txt urfinder.txt| sort -u | anew all\_urls.txt
