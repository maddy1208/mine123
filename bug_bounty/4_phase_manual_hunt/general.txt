#simple git check
Finding a .git directory is common. Exploiting it is where you get paid. Most tools fail if the directory returns a 403. git-dumper does not. It pieces the repo together from leaked fragments.

# How to actually dump a forbidden .git directory
python3 /tools/git-dumper.py https://target.com/.git/ ./output-dir

What I found: On a bug bounty program for a popular app, git-dumper reconstructed the entire source code from a forbidden .git folder. In a file called config.py.backup, I found hardcoded credentials for their cloud storage.
